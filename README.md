D db-vk
=====
D rewrite a [db-vk](https://github.com/scorpp/db-vk) deadbeef plugin for listening music from vk.com

# VERY EXPERIMENTAL (very buggy)
Created without knowledge dlang for learning dlang

Features
--------
* Retrieve 'My Music' contents
* Play
